Welcome to our blog! This is a sample post managed with AstroCMS.

## Getting started

AstroCMS lets you edit your Astro content collections from a visual interface. Browse your files, edit MDX content, manage frontmatter fields, and commit changes directly from the browser.

## Features

* Visual MDX editor with frontmatter form
* File tree browser for content collections
* Git integration (status, diff, commit, push)
* Astro component discovery
* Media upload
