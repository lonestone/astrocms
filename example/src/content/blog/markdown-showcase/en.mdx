## Comparison Table

| Feature         | AstroCMS           | Traditional CMS |
| --------------- | ------------------ | --------------- |
| Content format  | MDX files in Git   | Database        |
| Version control | Built-in (Git)     | Plugin required |
| Deployment      | Static or SSR      | Server required |
| AI assistant    | Claude integration | Not available   |
| Customization   | Full code access   | Theme-based     |

## Code Example

Here is a minimal Astro component:

```html
---
interface Props {
  title: string
  count?: number
}

const { title, count = 0 } = Astro.props
---

<div class="card">
  <h2>{title}</h2>
  <p>Count: {count}</p>
</div>
```

And a TypeScript helper:

```typescript
function getSlugFromId(id: string): string {
  return id.split('/')[0]
}

function getLangFromId(id: string): string {
  return id.split('/')[1]
}
```
