## Tableau comparatif

| Fonctionnalité    | AstroCMS              | CMS traditionnel    |
| ----------------- | --------------------- | ------------------- |
| Format de contenu | Fichiers MDX dans Git | Base de données     |
| Versioning        | Intégré (Git)         | Plugin nécessaire   |
| Déploiement       | Statique ou SSR       | Serveur requis      |
| Assistant IA      | Intégration Claude    | Indisponible        |
| Personnalisation  | Accès au code complet | Basé sur des thèmes |

## Exemple de code

Voici un composant Astro minimal :

```html
---
interface Props {
  title: string
  count?: number
}

const { title, count = 0 } = Astro.props
---

<div class="card">
  <h2>{title}</h2>
  <p>Total : {count}</p>
</div>
```

Et un helper TypeScript :

```typescript
function getSlugFromId(id: string): string {
  return id.split('/')[0]
}

function getLangFromId(id: string): string {
  return id.split('/')[1]
}
```
